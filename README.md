# **M2** 
## ***Loot Generator***

***Loot tier:***

* Tier1 - Worse loot
* Tier2 - MidLow loot
* Tier3 - MidHigh loot 
* Tier4 - Best loot

**Features:**

* Loot randomizer
* Loot spawn area
* Custom Loot Spawn(Select specific items to spawn)
* Loot Spread
* Special Events
* Loot Boxes(**Not Gamble, physical box**)
